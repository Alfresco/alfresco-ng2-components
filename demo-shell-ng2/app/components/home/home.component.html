Featured demos

Some of the areas will require diffent authentication providers. See details for each component. You may be automatically redirected to Login screen.

DocumentList

Demonstrates multiple Alfresco ECM components used together:
  • Document List ( ng2-alfresco-documentlist )
  • Upload ( ng2-alfresco-upload )
  • Context Menu ( ng2-alfresco-core )
  • DataTable ( ng2-alfresco-datatable )

Authentication provider: ECM

Activiti

Demonstrates multiple Alfresco BPM components used together:
  • App List ( ng2-activiti-tasklist )
  • Task List ( ng2-activiti-tasklist )
  • Process List ( ng2-activiti-processlist )
  • Form ( ng2-activiti-form )
  • Analytics ( ng2-activiti-analytics, ng2-activiti-diagrams )
  • DataTable ( ng2-alfresco-datatable )

Authentication provider: BPM
