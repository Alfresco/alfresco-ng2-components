# Toolbar Divider Component

Divides groups of elements in a Toolbar with a visual separator.

## Basic Usage

```html
<adf-toolbar>
    <button></button>
    <button></button>
    <adf-toolbar-divider></adf-toolbar-divider>
    <button></button>
</adf-toolbar>
```
